⌨️ Haxball Komutları Rehberi

Haxball'da hızlı yönetim, kontrol ve eğlence için kullanılan komutlar sayesinde sunucu yönetimi çok daha kolaydır. Komutlar genellikle ! veya / ile başlar ve sadece admin tarafından değil, bazıları tüm oyuncular tarafından da kullanılabilir.

📋 Temel Komutlar

  • !help → Kullanılabilir tüm komutları listeler.
  • !admin [isim] → Belirtilen oyuncuyu admin yapar.
  • !mute [isim] → Oyuncunun yazmasını engeller.
  • !unmute [isim] → Susturulan oyuncuyu tekrar aktif eder.
  • !ban [isim] → Oyuncuyu odadan kalıcı olarak atar.
  • !kick [isim] → Oyuncuyu o anlık oyundan atar.
  • !clearbans → Tüm banları temizler.
  • !pause / !unpause → Maçı durdurur / devam ettirir.

🛠️ Oda Ayarı Komutları (Bazı odalarda aktif)

  • /rec → Maçı kaydetmeye başlar.
  • /stop → Kayıt almayı durdurur.
  • /muteall → Herkesi susturur.
  • /unmuteall → Herkesi tekrar açar.

🧑‍⚖️ Admin Komutları

  • !spec [isim] → Oyuncuyu izleyiciye alır.
  • !red [isim] → Oyuncuyu kırmızı takıma alır.
  • !blue [isim] → Oyuncuyu mavi takıma alır.
  • !afk [isim] → Oyuncuyu otomatik olarak spec’e yollar.

🎮 Eğlenceli & Kullanışlı Ekstra Komutlar (Bazı modlarda)

  • !stats → Oyuncu istatistiklerini gösterir.
  • !rank → Sıralamayı gösterir.
  • !reset → Tüm ayarları sıfırlar.
  • !random → Oyuncuları rastgele takımlara ayırır.

💡 İpuçları

Komutlar çoğu zaman bot eklentileri veya özel scriptlerle aktif hale gelir. Eğer bazı komutlar çalışmıyorsa, büyük ihtimalle o odada aktif bot yoktur. Yine de !help komutunu yazarak desteklenen komutları görebilirsin.
